# CV Blog Admin Paneli

Bu repo, ASP.NET Web Forms kullanılarak geliştirilmiş bir admin panelini içerir.

## Proje Yapısı
- **`cv_blog_home.aspx`**: Blog ana sayfasının tasarımı.
- **`admin.Master`**: Admin paneli için Master Page.
